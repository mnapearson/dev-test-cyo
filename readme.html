Job Interview Task

To show us your skills and how you work with HTML, CSS and JavaScript, we want you to build a small landing page for spreadshirt.com. You will find 2 screens in the ZIP-file that illustrate how the page should look like on desktop and mobile. Please keep it simple and don't spend more than 4-5 hours.

If you feel the layout takes too much of your time leave some stuff out at the end and concentrate on the product slider

Things we want you to utilize/keep in mind:

  • Make use of HTML5 and CSS3 feature (e.g. picture-element, flexbox)
  • If you know SASS please show us
  • Make sure the page works with 360px width for mobile phones
  • Just use base fonts like Arial for everything
  • Don't copy the text manually from the images. Just use placeholder text
  • Upon page load dynamically create a slider showing 5 articles with JavaScript:
    • Use Swiper-Slider for the product carousel in the middle of the page (API Reference). Set it to show only 2 items on mobile and then gradually increase the number of items until 5 are displayed on desktop. (you can just leave out the arrows or use default ones from swiper)
    • For the products in the slider please use this API and pull the name, price and productId
    • the productId from the API can be used to display the image like so:
      https://image.spreadshirtmedia.com/image-server/v1/mp/products/productId/views/1,width=378,height=378,backgroundColor=F2F2F2
    • there may be images that don't work because they need special parameters. just use a different article then.
    • the API may output different products than on the mockup. don't worry about that.
  • Use vanilla JavaScript (no jQuery, React, Vue, etc.)
  • Browser-Support:
    • Safari 10.1+ (if you can't test it because you're on Windows ignore it)
    • Google Chrome 66+
    • Mozilla Firefox 60+
    • Internet Explorer 11 (if you can't test it because you're on Mac ignore it)
    • Microsoft Edge 16+ (if you can't test it because you're on Mac ignore it)
